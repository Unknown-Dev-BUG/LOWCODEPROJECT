# LowcodeProject (`lowcode-project`)

ยินดีต้อนรับสู่โครงการ **LowcodeProject** - ระบบจัดการและถอดความข้อมูลเอกสารอัตโนมัติด้วยระบบปัญญาประดิษฐ์ (AI Gemini)

---

## 🚀 ภาพรวมโครงการ (Project Overview)

โครงการนี้ถูกพัฒนาต่อยอดขึ้นมาโดยทีมงาน **Unknown-Dev-BUG** เพื่อตอบโจทย์ความต้องการใช้งานระบบจัดการฐานข้อมูลภายในองค์กรแบบปิด (Internal Local Environment) โดยสามารถนำเข้าและบันทึกเอกสารไฟล์ข้อมูลต่าง ๆ และให้ AI Gemini เข้าไปทำการถอดความ ดึงเนื้อหา และจัดเรียงความเกี่ยวข้องของเอกสารออกมาให้อยู่ในรูปแบบที่ต้องการโดยอัตโนมัติ

---

## 🛠️ โครงสร้างเทคโนโลยี (Technology Stack)

* **หน้าบ้าน (Frontend)**: React + Refine + Ant Design (`@refinedev/antd`)
* **ฐานข้อมูลภายใน (Database)**: Local Supabase (รันทำงานบน Docker ในวงแลนเครื่อง)
* **การเชื่อมต่อ AI (AI Processing)**: Google Gemini API (เชื่อมผ่าน REST SDK ล่าสุดอย่าง `@google/genai`)

## 🏁 ขั้นตอนเริ่มต้นทำงาน (Quick Start)

โปรดตรวจสอบให้มั่นใจว่าเครื่องของคุณรันระบบ **Docker Desktop** อยู่ก่อนเริ่มคำสั่งเหล่านี้:

1. **เปิดทำงานฐานข้อมูล Local Supabase**:
   ```bash
   supabase start
   ```
2. **ติดตั้งไลบรารีและรันเซิร์ฟเวอร์**:
   ```bash
   pnpm install
   pnpm dev
   ```
